Plugin to convert your website into a Real Estate Listing site using your properties on Wasi.co

== Description ==

This plugin will connect to the API of https://wasi.co to convert this websito into a Real Estate site using the properties located on your user account (using your API Credentials).
If you don't have your own API Credentials or don't know how to get them ยก, please contact us at soporte@wasi.co


== Installation ==

1. Upload `wasi-connector.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create your templates for listings, search results, featured listings, etc...
4. Add all shortcodes needed!
